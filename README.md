# SendMail
## Free email API
### How to use it ?
#### Base URL
https://mail.mat.run/api
#### Parameters
?name - Name of email sender

&receiver - Email receiver

&sub - Email subject

&message - Email message
#### Example
https://mail.mat.run/api?name=MAT.RUN&receiver=hi@mat.run&sub=hi&message=Hello%20World!
